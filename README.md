# 2048 Puzzle (учебный проект)

Классическая игра 2048 с консольным и графическим (Pygame) интерфейсом.  
Проект создан в учебных целях для отработки ООП, юнит-тестирования и базовых алгоритмов.

## Возможности

- Консольная версия (`matrix2048text.py`) – играйте через WASD.
- Графическая версия (`matrix2048graphics.py`) – с помощью Pygame.
- Юнит-тесты (запускаются через меню консольной версии, пункт 1).

## Как запустить

### Консольная версия
```bash
python matrix2048text.py
